What we're building (MVP)

Students can log in and see assignment grades and an overall course grade. Teachers can log in, add students, create assignments, and record grades. Multi-school ready.

  • Frontend: HTML/CSS (+ Bootstrap), vanilla JS
  • Backend: Node.js + Express, sessions
  • DB: SQLite (dev) → Postgres (prod) via Knex
How we work
  1. Plan by sprints (3 weeks each), assign tasks and track on Kanban.
  2. Code with linting and tests; review via PRs; deploy after green CI.
  3. Document decisions (ADRs) and risks; keep stakeholders informed.
